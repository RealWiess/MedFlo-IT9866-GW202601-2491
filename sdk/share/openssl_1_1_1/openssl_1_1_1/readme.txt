openssl_1_1_1w

目前只開啟CarPlay有使用到的option，一定需要的功能大致上有:
TLS1.2 - CarPlay 核心需求
AES - 加密核心演算法
SHA/SHA1/SHA256 - 基礎雜湊演算法


1. TLS相關
 [不需要/可關閉]
 TLS1      // 可關閉，太舊
 TLS1.1    // 可關閉，太舊
 TLS1.3    // 可關閉，CarPlay目前不需要
 SSL3      // 可關閉，已過時
 DTLS      // 可關閉，CarPlay不使用UDP
 DTLS1     // 可關閉，同上
 DTLS1.2   // 可關閉，同上

 [需要/不可關閉]
 TLS1.2         // 不能關閉，CarPlay需要


2. 加密演算法
 [不需要/可關閉]
 DES      // 可關閉，已過時
 RC4      // 可關閉，不安全
 RC2      // 可關閉，不安全

 [需要/不可關閉]
 AES      // 不能關閉，CarPlay需要


3. 雜湊演算法

 [需要/不可關閉]

 SHA1         // 不能關閉，CarPlay需要
 SHA256       // 不能關閉，CarPlay需要
